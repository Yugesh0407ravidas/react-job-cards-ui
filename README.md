# 🚀 React Job Cards UI

A modern frontend job portal built with **React.js** featuring reusable components, responsive design, and dynamic job data.

---

## ✨ Features

- 📌 Dynamic job listings with company logos  
- 🏷️ Job tags (Full-time, Part-time, Junior, Senior)  
- 💰 Pay details (hourly rates)  
- 📍 Location-based job display  
- ⚛️ Reusable React components  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6+)  
- HTML5  
- CSS3
